Data and code to create the manuscript version of this paper:

Taylor I, Boyd JK, McIndoo E and Ammons MCB (2026) A retrospective cohort study of viral and sociodemographic determinants of long COVID among Idaho veterans. *Front. Public Health* 14:1625363. [doi: 10.3389/fpubh.2026.1625363](doi: 10.3389/fpubh.2026.1625363)

- First run `modeling.R` to prepare data for modeling and fit the models reported in the paper.
- Then run `manuscript.qmd` to create the manuscript, figures, and tables.
